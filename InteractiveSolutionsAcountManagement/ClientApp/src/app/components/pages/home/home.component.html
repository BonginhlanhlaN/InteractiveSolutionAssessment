This is a single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code

What the website is about

The website solves a given problem, an accounts management problem. There a list of people in our system, each of the people has 0 or more accounts and every account has zero or more transactions.

Every one of these entities(People, Accounts and Transactions) can be Edited, Created and Deleted through our website.
